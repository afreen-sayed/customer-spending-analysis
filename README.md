# Customer Spending Score Analysis

## Description
This project analyzes customer spending patterns to understand purchasing behavior and trends.
It uses Python libraries **Pandas, Matplotlib, and Seaborn** for data cleaning, visualization, and analysis.

## Libraries Used
- pandas
- matplotlib
- seaborn
  
### Visualizations Included:
- Count Plot – Distribution of customers by Profession.
- Count Plot with Hue – Distribution of customers by Profession and Gender.
- Histogram – Distribution of Annual Income.
- Line Plot – Average Spending Score by Age (for Age > 20).
- Bar Plot – Average Spending Score by Profession.
- Scatter Plot – Relationship between Annual Income and Spending Score, colored by Gender.
- Heatmap – Correlation between numeric features like Age, Annual Income, Spending Score.
